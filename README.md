 # CancerDR: Cancer Drug Resistance Database

CancerDR was developed to address one of the major challenges in cancer treatment — acquired drug resistance. The database compiles pharmacological response data of anticancer drugs across diverse cancer cell lines along with mutation profiles, structural information, and sequence analysis of drug targets.

The resource supports personalized medicine approaches by helping researchers identify mutations responsible for drug resistance and discover drugs effective against specific cancer types.

Web Server: http://crdd.osdd.net/raghava/cancerdr/

## About the Database

CancerDR integrates pharmacological profiling data of anticancer drugs with mutation and structural information of drug targets.

The database contains:

- **148 anticancer drugs**
- **952 cancer cell lines**
- **116 drug targets**
- **1356 mutations**

Data were collected from resources such as:

- COSMIC
- CCLE
- PubChem
- UniProt

---

## Key Features

- Drug sensitivity profiling using IC50 values
- Mutation analysis of drug targets
- Predicted 3D structures of targets and mutants
- Sequence and structural alignment tools
- NGS read mapping and BLAST search
- Clustering of drugs and cancer cell lines

---

## Applications

CancerDR can be used for:

- Drug resistance analysis
- Personalized cancer therapy research
- Cancer genomics studies
- Structure-based drug discovery
